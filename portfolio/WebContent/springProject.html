1. 메인 페이지


속닥속닥의 메인 페이지 입니다.

  • 모임장이 등록한 다양한 모임 리스트를 볼 수 있습니다.

2. 모임 게시판


모임 게시판입니다.

  • 게시판에 들어가서 우측의 플러스 버튼을 누르면 게시물을 작성할 수 있습니다.

게시물 상세 보기

  • 게시물을 클릭하면 해당 게시물에 댓글을 달거나 좋아요를 표시할 수 있습니다.

3. 사진첩


모임 사진첩 입니다.

  • 스프링 프레임워크의 내장 객체인 CommonsMultipartResolver를 이용하여 파일 업로드 기능을 구현하였습니다.
  • 우측의 플러스 버튼을 클릭하여 사진을 추가할 수 있습니다.

사진 업로드 폼

  • 파일 선택 버튼을 클릭하여 여러개의 사진을 한꺼번에 올릴 수 있습니다.

사진이 업로드된 모습입니다.

  • 사진을 여러 장 올리더라도 사진첩에는 대표 사진 한 장만 보여지며
  • 게시물을 클릭하면 함께 올린 여러 장의 사진을 볼 수 있습니다.

4. 좋아요 기능


게시판과 사진첩의 좋아요 기능입니다.

  • Ajax로 구현하여 좋아요 버튼을 누르면 바로 좋아요수의 증가가 페이지에 반영됩니다.

게시판과 사진첩의 좋아요 기능입니다.

  • 좋아요를 누르면 즉시 좋아요수가 1 증가하며 꽉 찬 하트 모양으로 바뀝니다.

좋아요 취소

  • 한 회원당 한 게시물에는 한 번의 좋아요를 누를 수 있으므로
  • 다시 좋아요를 누르면 좋아요가 취소됩니다.

5. 모임 일정 관리


모임 일정 관리 기능입니다.

  • 모임 사이트의 좌측 하단에 있는 정모 만들기를 클릭하여 정모 일정을 등록할 수 있습니다.

정모 만들기 폼

  • 정모 날짜, 장소, 비용 등을 등록합니다.

일정(캘린더) 확인

  • 해당 모임의 모임원들은 마이페이지의 일정 한눈에 보기를 통해
  • 모임장이 추가한 정모 일정을 확인할 수 있습니다.

정모 참가하기

  • 모임원들은 모임 페이지에서 모임장이 등록한 정모에 참가 신청을 할 수 있습니다.
