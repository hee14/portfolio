1. 메인 페이지


쌍용시네마 메인 페이지

  • CGV, MEGABOX, 롯데시네마의 영화 상영 정보를 통합하여 제공하는 사이트입니다.

2. 로그인


로그인 페이지입니다.

  • MVC 패턴으로 구현하였습니다.

3. 회원가입


회원가입 폼

  • 아이디, 비밀번호, 이름, 성별, 휴대전화로 구성되어 있습니다.

4. 게시판(공지사항, Q&A)


게시판입니다.

  • 관리자 계정으로 로그인할 경우 공지사항에 글을 쓸 수 있습니다.

게시글 상세보기

  • 게시글을 수정하거나 삭제할 수 있습니다.

5. 영화 등록


영화 정보 등록

  • 관리자 계정으로 로그인시 상영하는 영화의 정보를 등록할 수 있습니다.
  • 영화 이름, 포스터 사진, 장르, 네이버 영화 링크를 입력하고 등록 버튼을 누릅니다.

6. 상영 스케줄 등록


상영 스케줄 등록

  • 관리자 계정으로 로그인시 영화의 상영 스케줄을 등록할 수 있습니다.
  • 상영 스케줄 등록을 원하는 영화를 선택하고 상영 시간, 가격, 상영관을 입력하고 등록 버튼을 누릅니다.

영화 등록 완료

  • 영화 소개 페이지에 등록된 것을 확인할 수 있습니다.

7. 영화 예매


영화 예매

  • 상영을 원하는 영화를 선택합니다.

좌석 선택

  • 인원수와 좌석을 선택한 후 예매하기 버튼을 누릅니다.

예매 완료 메시지

  • 예매가 완료되며 예매 확인 페이지나 예매 페이지로 이동할 수 있습니다.

예매 확인

  • 예매한 영화, 상영 날짜, 상영관, 좌석 번호를 조회할 수 있습니다.

8. CGV, 아트하우스의 실시간 영화 예매율


타 영화관 사이트의 영화 예매율입니다.

  • Jsoup을 사용하여 CGV와 아트하우스의 상영중인 영화 데이터를 크롤링하였습니다.
